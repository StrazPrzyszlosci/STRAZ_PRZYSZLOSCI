# ZLECENIE GŁÓWNE 100 - Safe automation orchestrator MVP

## Cel

Przygotować pierwszy bezpieczny szkic orkiestratora automatyzacji AI, który może uruchamiać tylko działania odwracalne: listowanie kolejek, generowanie sugestii, raporty smoke i drafty zadań.

## Kryteria odbioru

- Lista dozwolonych akcji MVP nie obejmuje zatwierdzania `approved`, eksportów produkcyjnych ani deployów.
- Każda akcja ma gate: wejście, uprawnienia, log/audit, rollback/no-op.
- Roadmapa wiąże boty Discord/Telegram, D1/SQLite, execution packi i handoffy.
- Dokument wskazuje pierwsze 3 komendy, które można bezpiecznie oddać automatyce.
